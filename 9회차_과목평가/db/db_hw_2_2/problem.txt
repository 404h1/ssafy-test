목표
학습주제
시스템을 위한 데이터베이스 설계와 구축
SQL 쿼리 작성 및 활용

학습목표
데이터베이스 스키마를 구성하는 방법과 데이터베이스 테이블을 생성하는 방법을 이해한다.
데이터를 삽입, 수정, 삭제하는 방법을 숙지한다.
데이터베이스에서 SELET 문을 사용하여 데이터를 조회하는 방법을 습득한다.

문제의도
시스템 설계를 위한 데이터베이스 설계와 SQL 쿼리 작성, 데이터베이스 조작 방법에 대해 이해한다.

문제
키오스크 제작 업체에서 주문 정보를 관리 할 수 있는 데이터베이스를 구축해달라는 요청을 받았다. 필요로 하는 데이터 유형과 구조는 이미지 파일로 넘겨 받았다. 제공된 이미지를 참고하여 요구 사항을 만족하는 데이터베이스를 구축하시오.
요구사항
product.sqlite3 파일을 생성하여 데이터베이스로 사용한다.
data.sql 파일을 생성한다.
SQL문은 생성한 data.sql 파일에서 작성하고 실행한다.
실행 방법
각 요구사항에 적절한 SQL문을 작성한다.
실행하고자 하는 SQL문을 우클릭한다.
Run Selected Query 를 선택하여 실행한다.
결과를 확인한다.

order
orders 테이블을 생성한다. 상세한 schema는 첨부 파일 참고.
orders 테이블에 서로 다른 데이터를 최소 3개 이상 삽입한다.
Date 타입은 'YYYY-MM-DD' 형식으로 삽입한다.
Real 타입은 10.11 형식으로 삽입 가능하다.

customers
customers 테이블을 생성한다. 상세한 schema는 첨부 파일 참고.
customers 테이블에 서로 다른 데이터를 최소 3개 이상 삽입한다.

데이터 수정
orders의 3번째 레코드를 삭제하시오.
customers의 1번째 레코드의 name을 '홍길동'으로 수정하시오.

데이터 조회
orders와 customers의 모든 데이터를 조회하시오.
