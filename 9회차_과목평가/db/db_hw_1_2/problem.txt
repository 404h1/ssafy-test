학습목표
SQL의 기본적인 개념과 역할에 대해 이해하고, 데이터베이스와 테이블의 구조를 파악할 수 있다. 
SQL을 사용하여 단일 테이블에서 데이터를 조회, 필터링하고 정렬, 제한할 수 있다.

문제의도
데이터베이스 스키마와 테이블 구조를 이해하고,  SELECT 문과 WHERE 문을 적절히 활용하여 원하는 데이터를 추출할 수 있다.

문제
주어진 db는 sqlite에서 문법 연습용으로 제공되는 DB이다. 하나의 DB에서는 다수의 테이블들을 관리 할 수 있는데, 이번에는 그 중 tracks 테이블을 사용하여 SQL 조회문을 연습해 보고자 한다. 요구 사항을 만족하는 코드를 작성하시오.
요구사항
data.sql 파일을 생성한다.
SQL문은 생성한 data.sql 파일에서 작성하고 실행한다.
실행 방법
각 요구사항에 적절한 SQL문을 작성한다.
실행하고자 하는 SQL문을 우클릭한다.
Run Selected Query 를 선택하여 실행한다.
결과를 확인한다.

tracks 테이블의 모든 데이터를 조회하시오.
Name, Milliseconds, UnitPrice 열의 모든 데이터를 조회하시오.
GenreId 행의 값이 1인 모든 데이터를 조회하시오.
모든 데이터를 name을 기준으로 오름차순 정렬하여 조회하시오.
tracks 테이블의 모든 데이터를 조회하되, 10개만 출력하시오.
